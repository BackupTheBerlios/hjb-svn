Provides the interfaces and classes used to process HttpServletRequests and send HttpServletResponses by generating {@link hjb.jms.cmd.JMSCommand}s, and sending appropriate responses back the HTTP user agent.

JMSCommandGenerator defines methods for matching a JMSCommand to a request, generating a command instance, and sending the appropriate response once the command is executed. There is an implementation of JMSCommandGenerator for each JMSCommand defined in the hjb.jms.cmd package

The JMSCommandResponders are used by the JMSCommandGenerator implementations to write the response back to the HTTP client

Other classes include:

  • JMSCommandGeneratorFactory provides JMSCommandGenerators for the relevant service methods of {@link hjb.http.HJBServlet}
  • HJBMessageWriter is used to write HJBMessages to the response as text
  • ParameterMapDecoder decodes parameters encoded in a HttpServletRequest
  • JMSArgumentFinder provides methods that look for arguments to the JMS API methods in the decoded parameters of a HttpServletRequest
