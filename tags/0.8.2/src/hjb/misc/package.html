Contains miscellaneous classes used throughout the various hjb subpackages.

HJBException, HJBClientException, HJBNotFoundException are runtime exceptions used to signal problems during processing of a {@link hjb.jms.cmd.JMSCommand}.

HJBStrings provides support for i18n and l11n across HJB. YetAnotherBase64 provides support for encoding/decoding byte arrays to Strings.
