Provides the classes used to allow values read and written from the fields of JMS MapMessages.

MapMessageValueCopier extends {@link hjb.msg.valuecopiers.BaseEncodedValueCopier} and serves as base class for the EncodedValueCopiers in this package. There are implementations to support reading and writing of boolean, char, byte, short, int, long, double, float, byte array and string field values from MapMessages.

OrderedMapMessageValueCopier combines all the other implementations and should be used when the type to be read/written is unknown.
