Contains the HJBServlet class and support classes.

HJBServlet is the entry point for all HJB requests. It dispatches requests made on URLs that HJB recognises, causing the invocation of JMS API methods on the JMS objects in the HJB runtime.

HJBServletConstants defines constants used in HJBServlet and its helper classes.

RuntimeExceptionHandler handles any exceptions that may occur while HJBServlet is processing requests.
