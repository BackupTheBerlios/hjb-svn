Provides the interface and classes used to allow data to be transferred between {@link hjb.msg.HJBMessage} and {@link javax.jms.Message} instances.

MessageCopier defines the methods used to copy data between the HJB and JMS versions of a message. The package contains implementations of this interface for each of the JMS Message types, all which extend a base class, PayloadMessageCopier which defines utility methods used by all the implementations.

MessageCopierFactory is used to instantiate the appropriate MessageCopier for a given message.

AttributeCopier, AttributeCopierAssistant and NamedProperyCopier are used by the implementations of MessageCopier to copy the attributes and properties between the two versions of a message.
